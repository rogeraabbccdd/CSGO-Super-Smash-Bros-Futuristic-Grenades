# CSGO Super Smash Bros Items: Futuristic Grenades
Add [Futuristic Grenades](https://github.com/Rachnus/Futuristic-Grenades) for [Super Smash Bros mode](https://github.com/rogeraabbccdd/CSGO-Super-Smash-Bros).

## Install
- Download plugin [here](https://github.com/rogeraabbccdd/CSGO-Super-Smash-Bros-Futuristic-Grenades/archive/master.zip).
- Download assets [here](https://github.com/rogeraabbccdd/CSGO-Super-Smash-Bros-Futuristic-Grenades/archive/assets.zip).
- Upload plugin to your game server.
- Upload assets to both your game server and fastdl server.
- Reload the map or restart server.
- Edit your configs in `cfg/sourcemod/futuristicgrenades.cfg`.
- Add items in `addons/sourcemod/configs/kento_smashbros/items.cfg`.

## Available Items
- item_blackhole
- item_forcefield
- item_explosion
- item_implosion
